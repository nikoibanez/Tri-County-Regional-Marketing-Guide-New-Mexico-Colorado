## Recommended Guide Structure

Add these under a practical "Post An Event" resource group:

1. Start local: county/town tourism or chamber forms.
2. Add media: World Journal, KRTN, KRCC/KUNM when audience fit makes sense.
3. Add arts-specific calendars: Southwest Contemporary, Colorado Fun Guide, PeakRadar only where coverage fits.
4. Add statewide tourism: Colorado.com or NMTD for visitor-facing tourism events.
5. Create a reusable public event URL: Facebook Event or Eventbrite.
6. Share into community groups only after checking rules.

## Notes For Public Copy

- Use "submit for review" or "request consideration," not "post guaranteed."
- Use "check group rules first" for Facebook groups.
- Use "verify dates, deadlines, and submission rules before relying on placement."
- Keep the local-first order: Trinidad / Huerfano / Angel Fire / Raton routes before statewide or broad platforms.
