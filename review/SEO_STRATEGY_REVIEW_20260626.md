# SEO Strategy Review

Date: 2026-06-26

Scope: Netlify static site generated by `tools/build_netlify_deep_guide.py`.

## Sources Checked

- Google Search Central SEO Starter Guide: https://developers.google.com/search/docs/fundamentals/seo-starter-guide
- Google title links guidance: https://developers.google.com/search/docs/appearance/title-link

## Implemented In The Generator

- Added indexable page metadata on every generated page: canonical URL, robots directive, theme color, Open Graph fields, Twitter preview fields, and JSON-LD `WebPage` structured data.
- Added site-level context to JSON-LD so each page is connected to the Stateline Guide site.
- Preserved unique page titles and strengthened page descriptions for the high-intent pages: Network, Amplifier Channels, About, and the home page.
- Kept `robots.txt` and `sitemap.xml`, with monthly change frequency and current `lastmod`.
- Added an alternate JSON link for the public guide data.
- Removed visible "confidence label" language from public pages and the app data bundle, replacing it with plain update/correction guidance.
- Kept correction/update pathways visible so outdated directory details have a clear feedback route.

## Recommended Search Positioning

Primary query intent:

- Tri-county business marketing guide
- Colfax Las Animas Huerfano business resources
- Raton Trinidad Walsenburg business directory
- Southern Colorado Northern New Mexico nonprofit resources
- Regional event calendar submission and promotion resources
- Creative business and artist resources Colfax Las Animas Huerfano

Content positioning:

- The home page should rank as the general regional guide.
- Network should rank for directory, chamber, tourism, business, nonprofit, artist, and support-resource searches.
- Amplifiers should rank for event calendars, newsletters, visitor guides, media, and promotional submission paths.
- Submit should rank less aggressively but remain findable for corrections and new listings.
- About should establish the guide as a practical routing manual, not a replacement for official offices or chambers.

## Ongoing SEO Rule

Do not chase keywords by adding generic filler. Each new page should answer a practical regional task: where to list, where to post, who to contact, what to prepare, and what to confirm before acting.
