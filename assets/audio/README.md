# Site Audio Sources

The music-bar tracks are public-domain regional field recordings from the Library of Congress Juan B. Rael Collection, formally titled *Hispano Music and Culture of the Northern Rio Grande: The Juan B. Rael Collection*.

## Regional Archival Tracks

- `loc-rael-nm-valse.mp3`
  - Title: "Valse de Cadena"
  - Performers listed by LOC recording log: Nieves Anaya and Rubel R. Cobos
  - Place: Arroyo Hondo, New Mexico
  - Collection: AFC 1940/002, Library of Congress
  - Source audio: https://tile.loc.gov/storage-services/service/afc/afc1940002/afc1940002_3914a1.mp3

- `loc-rael-co-valse.mp3`
  - Title: "Valse de Alejo Torres" / "Valse de Alejo Torreo" as listed in LOC search metadata
  - Performer listed by LOC recording log: Jose Rael
  - Place: Antonito, Colorado
  - Collection: AFC 1940/002, Library of Congress
  - Source audio: https://tile.loc.gov/storage-services/service/afc/afc1940002/afc1940002_3918a1.mp3

LOC item pages for this collection state that the Library understands the 1940 folk music recorded by Juan B. Rael to be within the public domain. Keep this note with the site package so future maintainers can preserve the source trail.
